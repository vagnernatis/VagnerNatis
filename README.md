## 🌱 Sobre minha jornada

Não estou com pressa de saber de tudo — meu objetivo é construir uma base **sólida**, mesmo que em um ritmo mais lento. Escolhi começar pelo Desenvolvimento Web como ponto de partida, justamente porque a área de tecnologia é enorme, e antes de me espalhar entre vários caminhos, decidi me aprofundar em um primeiro.
